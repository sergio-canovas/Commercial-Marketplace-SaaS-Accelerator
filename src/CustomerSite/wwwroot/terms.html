2. Description of the Product

The dForts Copilot Readiness Monitor is a SharePoint Framework (SPFx) application that helps Microsoft 365 organizations identify and remediate oversharing risks in SharePoint Online sites prior to or during deployment of Microsoft 365 Copilot. The Product consists of:

  • Site-level web part — installed by site owners to analyze and remediate oversharing in a specific SharePoint site.
  • Tenant-wide report web part — installed by SharePoint Administrators to obtain an organization-wide oversharing overview across all sites.
